# PowerStack NodeJS Indexer Starter

NodeJS Indexer Starter Template

⚡️ [PowerStack: a powerful fullstack development framework](https://powerstack.xyz).

## Yarn

```
# Install project dependencies
yarn

# Development server with reload
yarn dev

```
